## 4. 当前判断

1. `ldrc_adaptive` 40k 是当前综合最优结果，成功率最高、碰撞率低、最终误差最低。
2. `ldrc_fixed` 50k 与 `ldrc_adaptive` 40k 接近，且 jerk 明显更低，说明约束 SAC + 连杆风险是主要收益来源。

4. `ee_fixed` 100k 综合表现弱于 LDRC 方法，说明只使用末端风险不足以处理连杆碰撞。
